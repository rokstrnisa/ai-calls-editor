# Contributing to AI-calls-Editor

Thank you for your interest in contributing!

## Development Setup

1. Fork and clone the repository.
2. Install dependencies:
    ```bash
    pnpm install
    ```
3. Start watch mode for auto-compilation:
    ```bash
    pnpm run watch
    ```
4. Press `F5` in VS Code to launch the Extension Development Host.

## Making Changes

- Keep the codebase minimal and readable.
- Run `pnpm format` before committing.
- Test your changes in the Extension Development Host.

## Submitting Pull Requests

1. Create a descriptive branch name (e.g., `fix-rename-edge-case` or `add-error-handling`).
2. Make your changes with clear commits.
3. Ensure all existing functionality still works.
4. Push to your fork and submit a pull request.
5. Describe what your PR does and why.

## Code Review

Maintainers will review your PR and may request changes. We prioritize:

- Code clarity and simplicity.
- Minimal dependencies.
- Well-tested functionality.

## Questions?

Open an issue for discussion before starting work on significant changes.
